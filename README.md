# 🚀 NIFTY50 PULSE

A futuristic cyberpunk-themed NIFTY 50 stock market dashboard built using pure HTML, CSS, and JavaScript.

NIFTY50 PULSE provides real-time market simulation, portfolio management, watchlists, stock screening, options analysis, AI-powered market assistant, risk analysis, crash prediction, and paper trading in a single responsive web application.
